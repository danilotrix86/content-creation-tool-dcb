# content-creation-tool-dcb

AI-powered article generation with competitor research, SEO optimization, and visual assets.

## Project structure

- `web/` — Next.js app (generation UI and API)
- `supabase/migrations/` — Database schema migrations

## Setup

See [web/README.md](web/README.md) for environment variables and local development.

```bash
cd web
npm install
cp .env.example .env.local
npm run dev
```

Open [http://localhost:3000](http://localhost:3000).
